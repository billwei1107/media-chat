# PaiPai Android/iOS App

This directory is reserved for the Flutter application.

> [!WARNING]
> The `flutter` command was not found in the system path during initialization.

## Setup Instructions
1. Install Flutter SDK: [https://docs.flutter.dev/get-started/install](https://docs.flutter.dev/get-started/install)
2. Run the following command in this directory:
   ```bash
   flutter create . --project-name=paipai_app --org=com.paipai
   ```
3. Open the project in your preferred IDE (VS Code / Android Studio).
