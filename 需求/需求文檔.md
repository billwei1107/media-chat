# 派派连萌 (PaiPaiLianMeng) - 需求文檔

## 1. 項目概況
**項目名稱**: 派派连萌
**項目目標**: 打造一個集社交、短視頻、即時通訊於一體的綜合性社交平台，提供豐富的互動體驗。

## 2. 用戶角色
- **普通用戶**: 瀏覽內容、發布動態、添加好友、聊天、管理個人資料。
- **管理員** (隱含): 內容審核、用戶管理 (舉報處理)。

## 3. 功能需求

### 3.1 認證與 Onboarding (Authentication & Onboarding)
| 功能模組 | 描述 |
| :--- | :--- |
| **啟動頁與引導** | 展示核心功能優勢，引導用戶完成初始操作。 |
| **登入** | 支援 **郵箱**、**微信**、**QQ** 等多種社交帳號登入。 <br> 提供 **手機驗證碼** 找回密碼與快捷登入。 |
| **註冊** | 填寫用戶名、綁定 **郵箱/手機**、設置安全密碼。 |
| **信息蒐集** | 讓用戶自主選擇感興趣的 **社交主題標籤** (信息蒐集)。 <br> 詢問是否允許訪問 **通訊錄** 以添加好友。 |

### 3.2 首頁與動態 (Home & Feed)
| 功能模組 | 描述 |
| :--- | :--- |
| **通知** | 實時展示系統通知、好友互動等通知信息。 |
| **主播推薦** | 依據用戶偏好推薦優質主播，方便關注。 |
| **首頁推薦** | 精準為用戶推薦匹配興趣的博主，促進關注。 <br> 展示感興趣內容，支援 **點讚、評論、轉發、收藏**。 |
| **為你推薦** | 可選擇 **生活、娛樂、美食** 等不同分類查看對應推薦內容。 |
| **創建帖子** | 支援上傳 **文字、圖片、視頻** 等多種形式內容。 <br> 提供 **話題標籤** 添加功能，提升帖子曝光度。 |

### 3.3 發現 (Discovery)
| 功能模組 | 描述 |
| :--- | :--- |
| **多樣化話題** | 提供多樣化話題入口，助力用戶發現興趣內容。 |
| **熱門活動** | 展示熱門活動與趨勢，引導用戶參與互動。 |

### 3.4 觀看直播 (Live Streaming) [NEW]
| 功能模組 | 描述 |
| :--- | :--- |
| **實時觀看** | 實時觀看主播直播，支援發送 **彈幕** 互動。 |
| **禮物贈送** | 提供 **禮物贈送** 功能，增強與主播互動性。 |

### 3.5 我的 (Profile)
| 功能模組 | 描述 |
| :--- | :--- |
| **個人資料** | 查看與編輯個人頭像、簡介等資料。 |
| **管理帖子** | 管理已發布帖子，支援 **編輯、刪除** 操作。 |
| **賬戶設置** | 進行賬戶安全、隱私等賬戶設置調整。 |
| **切換賬戶** | 支援添加多個賬號，便捷切換登錄賬戶。 |

## 4. 非功能需求 (Non-functional Requirements)
- **多語言支援**: 根據 UI (06-語言選擇) 必須支援多語言切換。
- **響應式體驗**: 視頻播放需流暢，轉場動畫需順滑。
- **安全性**: 所有用戶數據需加密傳輸，密碼需雜湊存儲 (依循全局規則)。

## 5. 待確認項目
- **後端架構**: 需確認使用之技術棧。
- **第三方服務**: 短信驗證碼服務商、地圖服務 (若有定位功能)、雲存儲服務商。
